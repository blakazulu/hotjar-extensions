Privacy Policy

We respect your privacy and don't collect any data

Last Updated: November 4, 2025

Overview

Hotjar Blocker is a browser extension that blocks Hotjar tracking requests on domains specified by the user. We prioritize user privacy protection above all else.

Data Collection

We do not collect any data. The extension does not gather, retain, or share any personal information or user data with external parties. Period.

Local Storage

Only domain names you designate for protection are stored locally through your browser's storage system. These may sync across your devices if you're signed into your browser (Chrome Sync or Firefox Sync).

This data never leaves your browser and is never transmitted to our servers or any third party because we don't operate any servers for this extension.

What We Don't Collect

The extension does not collect:

  • Personal information (name, email, address, etc.)
  • Browsing history or activity
  • Website content you view
  • User behavior or analytics
  • IP addresses or location data
  • Any information from websites you visit
  • Blocked request details (we only count them locally)

Permissions Explanation

The extension requests specific permissions to function properly:

  • declarativeNetRequest / webRequest: Required to block Hotjar network requests
  • storage: Required to save your protected domain list locally in your browser
  • tabs: Required to display blocking status for the current tab
  • <all_urls>: Required to monitor requests on all websites (only blocks Hotjar domains)

These permissions are used exclusively for the extension's functionality and do not enable any data collection.

Third Parties

We do not share, sell, or transfer any data to third parties because we don't collect any data in the first place. There are no analytics services, advertising networks, or any external services integrated into this extension.

Open Source

Hotjar Blocker is open source software. You can review the complete source code on GitHub to verify our privacy claims.

Changes to This Policy

If we make any changes to this privacy policy, we will update the "Last Updated" date at the top of this page. We recommend reviewing this policy periodically.
